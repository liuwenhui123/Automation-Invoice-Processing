# 发票自动化处理工具

一个智能的发票自动化处理工具，支持PDF发票解析、批量重命名、多分类归档和Excel导出。

## ✨ 主要特点

- **智能解析**：自动解析PDF发票，提取关键信息
- **批量处理**：支持批量重命名和归档
- **多分类归档**：一张发票可归入多个类别
- **Excel导出**：按标准格式导出发票信息
- **直观界面**：
  - 每个发票下直接显示类别复选框
  - 工具栏批量归类单选按钮
  - 实时分类汇总统计
- **文件优化**：移动文件而非复制，节省磁盘空间
- **自动关闭**：执行完成后自动关闭窗口

## 🚀 快速开始

### 方式1：直接运行EXE（推荐）
```
双击 dist\invoice_renamer.exe
```

### 方式2：使用启动脚本
```
双击 start.bat
```

### 方式3：Python运行
```bash
python invoice_renamer.py
```

## 📋 使用流程

1. **启动程序** → 自动扫描当前目录PDF文件
2. **查看发票** → 每个发票下方显示类别复选框
3. **归类方式**：
   - **单个归类**：直接勾选发票下方的复选框
   - **批量归类**：点击工具栏的类别单选按钮
4. **管理类别** → 点击"⚙️ 管理类别"按钮
5. **执行归档** → 点击"✅ 开始执行"
6. **自动完成** → 显示成功提示后自动关闭

## 📊 Excel导出格式

导出的Excel文件包含以下列：
1. 发票号码
2. 开票日期
3. 购买方名称
4. 购买方税号
5. 销售方名称
6. 销售方税号
7. 发票金额
8. 发票税额
9. 发票总额
10. 总额大写
11. 物品名称

## 💡 主要功能

### 文件处理优化
- **移动而非复制**：节省磁盘空间
- **智能处理多类别**：
  - 无类别：重命名后保留在原目录
  - 单类别：移动到该类别目录
  - 多类别：移动到第一个类别，其他类别复制副本

### 批量归类
- 工具栏直接显示类别单选按钮
- 点击即可将所有发票归类到该类别
- 无需弹出对话框，操作更快捷

### 自动关闭
- 执行成功后自动关闭窗口
- 有错误时保持打开让用户查看

## 🔧 开发说明

### 环境要求
- Python 3.8+
- Windows 10/11

### 依赖库
```bash
pip install pypdf openpyxl
```

### 打包为EXE
```bash
.\build_exe.ps1
```

或使用PyInstaller：
```bash
python -m PyInstaller --onefile --console --name "invoice_renamer" invoice_renamer.py
```

### 项目结构
```
invoice/
├── invoice_app/
│   ├── __init__.py
│   ├── models.py              # 数据模型
│   ├── parser.py              # PDF解析引擎
```

## ⚠️ 注意事项

1. **执行归档后原文件会被移动**，请先备份重要文件
2. **多类别处理**：第一个类别移动原文件，其他类别复制副本
3. **建议先测试**：在测试目录试用后再处理正式文件
